# Khareedo Support Bot 🤖

Khareedo Support Bot is an AI-powered customer support chatbot developed using **Dialogflow ES** for the Khareedo e-commerce platform.

This chatbot is designed to assist customers by answering common store-related queries automatically, improving customer support and enhancing user experience.

--

## 📌 Features

- Greeting and welcome users
- Help/Support guidance
- Product inquiry handling
- Pricing information
- Shipping details
- Return/Refund policy assistance
- Contact information support
- Goodbye/Farewell responses

---

## 🧠 AI Concepts Used

- **Intents** for understanding user requests
- **Training Phrases** for natural language recognition
- **Entities** for identifying products/categories
- **Contexts** for maintaining conversation flow
- **Fallback Intent** for unrecognized queries

---

## 🛠️ Technologies Used

- **Dialogflow ES**
- Natural Language Processing (NLP)
- Conversational AI Design
